图的深度与广度优先遍历

发表于2022-09-18|更新于2022-09-19|算法
|字数总计:325|阅读时长:1分钟|阅读量:

深度优先遍历算法口诀

  • 访问根节点
  • 对根节点的没访问过相邻节点挨个进行深度优先遍历
    code

    广度优先遍历算法口诀

  • 新建一个队列，把根节点入队
  • 把队头出队并访问
  • 把队头的没访问过的相邻节点入队
  • 重复第二三步，直到队列为空
    code 执行结果

总结

  • 图是网络结构的抽象模型，是一组由边连接的节点
  • 图可以表示任何二次元关系，比如道路，航班
  • JS中没有图，但是可以用，Object和Array构件图
  • 图的表示法：邻接矩阵,邻接表
  • 图的常用操作: 深度/广度优先遍历
